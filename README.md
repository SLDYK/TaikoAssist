# TaikoAssist

太鼓达人练习辅助器，一个基于 Unity 的个人练习项目。

## 项目目标

- 提供便捷的节奏练习场景
- 尝试做一些节奏训练相关的小工具
- 作为个人 Unity 练手项目长期迭代

## 技术栈

- Unity（URP 2D 模板）
- C#
- Input System

## 运行环境

- Windows 10/11（已在 Windows 环境下开发）
- Unity Editor（建议使用与项目匹配的版本打开）

## 快速开始

1. 克隆仓库。
2. 使用 Unity Hub 添加本项目目录。
3. 用 Unity 打开项目，等待依赖导入完成。
4. 在 `Assets/Scenes/` 中打开场景并运行。

## 目录结构

- `Assets/`：游戏资源、脚本和场景
- `Assets/Scripts/`：核心脚本
- `Assets/Scenes/`：主要场景
- `Packages/`：Unity 包管理配置
- `ProjectSettings/`：项目设置

## 开发说明

- 仓库已配置 Unity 常用忽略规则，避免提交 `Library/`、`Temp/` 等生成目录。
- 如果你修改了项目设置，Unity 可能会更新 `ProjectSettings/` 下文件，属于正常现象。
